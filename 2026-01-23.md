\# 2026-01-21 / Q02 — 멀티 리전 DR(최소 다운타임) 설계



\## 1) 문제를 쉬운 말로 재해석

* 새로운 CloudFront 생성 중이다.
* 그런데 사용자가 입력하는 정보 중에 민감한 정보가 있다.
* 이 정보는 전송 중뿐만 아니라 서버까지 가는 전체 과정에서 보호되어야 한다.
* 그리고 아무 애플리케이션이나 이 정보를 볼 수 있으면 안 되고 특정 애플리케이션만 볼 수 있어야 한다.



\## 2) 알게된 개념

* CloudFront 서명된 URL 구성 : 특정 url 에 대해 정해진 시간 조건에서만 접근을 허용하는 url
* CloudFront 서명 쿠키 : 브라우저에 접근 허가증 쿠키를 심어서 특정 경로/파일들을 일정 시간 동안만 접근 가능하게 하는 기능
